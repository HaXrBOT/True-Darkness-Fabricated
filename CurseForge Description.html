This is an unofficial fork of the True Darkness mod by GrondagTheBarbarian, bringing it to Fabric 1.20.1 and above.
For a Forge fork, see here.

GrondagTheBarbarian is the original author and all credit belongs to him.

If the original mod gets updated, this page will be archived.

 

Description:

Moody brightness not dark enough? Do you want moonless nights and unlit caverns to be pitch black? This mod is for you!
 
In addition to porting the mod to newer versions, this fork also fixes compatibility issues with Distant Horizons.

 

Dependencies:

Required: Fabric API
 
Optional: Mod Menu
 
Only required client-side. Does not have any functionality when installed on a server.
